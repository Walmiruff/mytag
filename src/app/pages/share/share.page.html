Share API

The Share API provides methods for sharing content in any sharing-enabled apps the user may have installed. The Share API works on iOS, Android, and the Web (using the new Web Share API).

Currently this feature is only available on mobile browser that has Web Share API.

We also use the Device API to get your Device info.
