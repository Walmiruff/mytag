PWA Progressive Web Apps (PWA)

The PWA can run in any browser and can be installed on your desktop and mobile phone It will act exactly like any native apps, without the need to download from app store We want to make it easier for you to get started with PWA development. The template scores 100% for PWA on Lighthouse.

Follow our easy, step-by-step instructions, and within minutes, you will have a PWA similar to this, hosted on your firebase hosting.

Add to Home Screen, sometimes referred to as the web app install prompt, makes it easy for users to install your Progressive Web App on their mobile or desktop device. After the user accepts the prompt, your PWA will be added to their launcher, and it will run like any other installed app.
