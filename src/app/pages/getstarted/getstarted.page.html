Get Started
1. Platform Setup

  • Make sure you have an up-to-date version of Node.js installed on your system. If you don't have Node.js installed, you can install it from

    https://nodejs.org/en/download

  • Open a terminal window (Mac) or a command window (Windows), and install Cordova and Ionic:

    npm install -g cordova ionic

    On a Mac, you may have to use sudo depending on your system configuration:

    sudo npm install -g cordova ionic

    If you already have Cordova and Ionic installed on your computer, make sure you update to the latest version:

    npm update -g cordova ionic

    OR

    sudo npm update -g cordova ionic

  • Install your code editor such as Visual Studio Code or Atom
2. Set-up Firebase

  • Sign-up for Firebase

    https://firebase.google.com/

  • Go to Firebase Console

    https://console.firebase.google.com

  • Add your Project

  • Get you Firebase configuration file

    Go to Project Settings/General and click Add Firebase to your Website icon

    Copy the configuration file. You will need it later to paste in your src/apps/credentials.ts file when you install and setup the template in Item 4 below.
3. Set-up Authentication

  • Go to Firebase Console

    https://console.firebase.google.com

  • Go to Authentication/Sign-in method

  • Enable Google

    For Google authentication, it will be setup automatically.

  • Enable Facebook

    Go to Facebook Developer page to get your App ID and App Secret

    https://developers.facebook.com/apps/

    Add the OAuth redirect URI (from Firebase) to your Facebook app configuration.

    For information on how to setup your facebook apps please refer to:

    https://developers.facebook.com/docs/apps/

    Update your facebook App ID and App Secret in firebase
4. Install and Set-up Template

  • Unzip the Template file to your project directory

  • Open a terminal window (Mac) or a command window (Windows), and navigate to the project directory.

  • Install the server dependencies:

    npm install

  • Open your project using Visual Studio Code or Atom

    Open your Firebase Configuration file at src/apps/credentials.ts

    Follow the instruction in the credentials.ts file

  • Test it in your browser.

    ionic serve

  • If you follow all the Steps from 1 - 4, your apps will be displayed in your browser.
5. Host it on Firebase Hosting

  • Open a terminal window (Mac) or a command window (Windows) and go to your project directory

  • Install the Firebase CLI.

    npm install -g firebase-tools

  • Initialize your site

    firebase init

  • On subsequent build and deploy, you do not need to do the above 2 steps.

  • Build your project

    ionic build --prod

  • Deploy your site. Upon completion, go to the URL provided.

    firebase deploy

  • CONGRATULATION

    You Apps is now LIVE.
